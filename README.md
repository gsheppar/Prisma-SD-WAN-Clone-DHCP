# Prisma SD-WAN DHCP Clone (Preview)
The purpose of this script is to clone DHCP subnets from one site to another

#### License
MIT

#### Requirements
* Active CloudGenix Account - Please generate your API token and add it to cloudgenix_settings.py
* Python >=3.7

#### Installation:
 Scripts directory. 
 - **Github:** Download files to a local directory, manually run the scripts. 
 - pip install -r requirements.txt

### Examples of usage:
 Please generate your API token and add it to cloudgenix_settings.py
 
 1. ./clone_dhcp.py -S Branch-Site-1 -D Branch-Site-4
      - Will copy all DHCP Subnets from one branch to another 

### Caveats and known issues:
 - This is a PREVIEW release, hiccups to be expected. Please file issues on Github for any problems.
